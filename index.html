HTML e CSS:

HTML (HyperText Markup Language) é a linguagem padrão usada para criar e projetar páginas da web.
Ela utiliza um sistema de tags para estruturar conteúdo na web, tornando-o legível e interpretável por navegadores da web.

Principais tags html:
Title
Body
H1
Img
A

CSS (Cascading Style Sheets) é uma linguagem de folha de estilo usada para descrever a apresentação e o layout de um documento escrito em HTML ou XML.
O CSS é essencial para controlar a aparência das páginas da web e separar o conteúdo do design.

Principais estilos css:
Color
Font Family
Margin
Text-Align
Padding
